## VSCode
- .vscode  
    Mac: /Users/[USER NAME]/.vscode  
    Windows: C:\Users\ユーザー名\AppData\Roaming\Code\User
- settings.json  
    Mac: /Users/[USER NAME]/Library/Application\ Support/Code/User  
    Windows: C:\Users\ユーザー名\AppData\Roaming\Code\User\settings.json
